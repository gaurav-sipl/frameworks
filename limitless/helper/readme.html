6Common Points

Common points need to remeber while devlopment
  1. Wrap all js files with IIFE functions

    it will helpful to reduce chances of conflictions

  2. Make sure all Parent methods name should be Capitalize and child should be in camelCase
  3. Follow Bootstrap structure as shared in grid above (see section 5).
  4. Please do not any garbage in code, we should keep all code clean and clear.
  5. Functions names should be more convinient & easy to understand i.e. CustomerListCtrl or CustomerListFactory.
  6. Please do not add any unnecessary files in folder, just use relavent files.
  7. Happy to help you 24/7/365!!!!!
